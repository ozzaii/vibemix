    (c) For each of the 20 animation GLBs, invokes `@gltf-transform/cli` to (i) drop mesh/material/texture nodes (`gltf-transform prune` + `gltf-transform dedup` keeping only skeleton + animation tracks), then Draco-compress the remaining buffer with `gltf-pipeline --draco.compressionLevel 10`.     (e) After all GLBs land, computes `du -sb tauri/ui/assets/mascot/` and asserts <= 26214400 bytes; if exceeded, prints per-file sizes sorted descending and exits non-zero with hint "consider --draco.quantizePositionBits 12 or strip more aggressively".
    (f) Idempotent — re-running on already-built output produces byte-identical files (`gltf-pipeline` is deterministic; `gltf-transform` is deterministic given the same node version).

    Add dev dependencies to `tauri/ui/package.json`: `gltf-pipeline@^4.1.0` and `@gltf-transform/cli@^4.0.0`. Add an npm script: `"build:mascot": "node scripts/build-mascot-bundle.mjs"`.
