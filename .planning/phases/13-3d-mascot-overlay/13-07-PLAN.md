The renderer holds a `currentMood` ref. When event-dispatcher (Plan 13-06) needs to translate `phase=groove` to a state, it asks `mood.ts pickFromPool(MOOD_PROFILES[currentMood].idle_pool)` â€” different moods pick differently.

ParticlePuff effect (THREE.Points):
- ~50 particles spawned at mascot torso position (head-of-rig + 0.3 down)
- Outward radial velocity, slight upward bias, fade-out alpha over 500ms
- Texture: single 32x32 circular soft alpha sprite generated procedurally via Canvas2D + THREE.CanvasTexture (no extra asset bundle)
- Material: `THREE.PointsMaterial({ map, transparent: true, depthWrite: false, blending: THREE.AdditiveBlending, size: 0.12, sizeAttenuation: true })`
- Color tinted by destination mood (hype = warm amber, teacher = soft cream, coach = cool slate). Use existing tokens.css palette equivalents â€” fetch from CSS variables OR hardcode the THREE.Color constants documented inline with reference to the design system.
