# Phase 20 Plan 05: __main__.py runtime wiring — anti-slop activation

## Objective

Close the runtime activation gap flagged by `20-VERIFICATION.md` (`gaps[0]`). The
Phase 20 anti-slop library is **shipped + tested** but the live binary
(`src/vibemix/__main__.py`) constructs `DJCoHostAgent` without the four wired
primitives (`citation_linter / stripped_rate_tracker / ack_bank / playback`) and
constructs `coach_loop(...)` without the two anti-slop kwargs
(`ipc_bus / citation_telemetry`). As a result, `_linter_wired` evaluates `False`
at runtime → the linter never fires against real Gemini responses, and the
`ipc.session.citation` publish gate is a no-op.

This plan:

1. Constructs `CitationLinter()` + `StrippedRateTracker(window_s=..., threshold=...)`
   + an `EvidenceRegistry()` (required for the linter to have a non-None
   snapshot — otherwise the linter would strip every response with reason
   `no_citations`) at startup in `__main__.py`.
2. Threads `evidence_registry` into `state_refresh_loop(...)` AND
   `DJCoHostAgent(..., evidence_registry=registry, citation_linter=linter,
   stripped_rate_tracker=tracker, ack_bank=ack_bank, playback=playback)`.
3. Constructs a tiny `CitationIpcShim` (in-process buffered shim that satisfies
   `await ipc_bus.emit(dict)` duck-type — appends each emitted dict to a
   bounded `collections.deque`) and threads it + a `citation_telemetry()`
   callable closure into `coach_loop(...)`.
4. Adds a `VIBEMIX_ANTI_SLOP` env-var feature flag (default `on`). When set to
   `"off"` / `"0"` / `"false"`, the linter wiring is bypassed and the legacy
   non-wired path runs — preserves the v4 byte-identity contract for callers
   that need it and gives Kaan a one-flag rollback if the live ear-test goes
   sideways.

The `CitationIpcShim` is intentionally an **in-process buffer**, NOT a websocket
emitter — wiring the bytes to the actual mascot WS clients (which run under
`ws_broadcast` on the same `127.0.0.1:8765` port) is a v2.x follow-up that
needs to either (a) refactor `ws_broadcast` to expose its `clients` set, or
(b) introduce a multiplexing bus class. Both are larger surgery than Phase 20
scope — the shim closes the kwarg gate `if citation_wired:` so the publish path
fires, the `coach_loop` records to the recorder, and the v2.x follow-up is
fully isolated.

**Wire `CitationIpcShim` + `citation_telemetry` callable into `coach_loop(...)`.**

<behavior>
1. New module `src/vibemix/coach/citation_ipc_shim.py` exports
   `CitationIpcShim`. The class has:
   - `__init__(self, maxlen: int = 64) -> None` — owns a
     `collections.deque[dict]` with the given maxlen.
   - `async def emit(self, msg: dict) -> None` — appends `msg` to the deque
     (no I/O, no await beyond function-coro boundary).
   - `def snapshot(self) -> tuple[dict, ...]` — returns a frozen tuple snapshot
     of buffered messages for introspection / tests.
   - `def __len__(self) -> int` — buffered count.
2. `__main__.py` imports `CitationIpcShim` from `vibemix.coach.citation_ipc_shim`.
3. `__main__.py` constructs `CitationIpcShim()` at startup when
   `anti_slop_enabled` is True.
4. `__main__.py` defines a `_citation_telemetry()` closure that returns a dict
   with the four keys `coach_loop` reads:
   - `"slop_ratio"` — derived from `evidence_registry.citation_telemetry()`
     (uses `1.0 / max(1.0, mean)` as a placeholder until v2.x defines the true
     slop metric — documented in SUMMARY).
   - `"stripped_rate_15s"` — `tracker.rate()` fresh per call.
   - `"last_unverified_response"` — `None` (v2.x follow-up — no simple
     existing source; ring-buffer wiring deferred).
   - `"bypass_active"` — `tracker.should_bypass()` NO — that's destructive
     (consumes the latch). Use `tracker._bypass_consumed` read-only?
     No — that's private. Use a non-destructive proxy:
     `tracker.rate() > STRIPPED_RATE_THRESHOLD` as the "would-bypass-if-asked"
     signal. This matches the docs in the SessionCitation schema better than
     a destructive `should_bypass()` call.
5. `coach_loop(...)` call (line ~494) receives `ipc_bus=citation_shim,
   citation_telemetry=_citation_telemetry` when enabled; both `None` when
   disabled.
6. Source-grep test assertions over `__main__.py`:
   - `from vibemix.coach.citation_ipc_shim import CitationIpcShim`
   - `CitationIpcShim(` constructor call
   - `ipc_bus=` kwarg on `coach_loop` call
   - `citation_telemetry=` kwarg on `coach_loop` call
7. Unit tests of `CitationIpcShim`:
   - Empty shim returns `()` snapshot.
   - `await shim.emit({"a": 1})` then `snapshot()` returns `({"a": 1},)`.
   - 100 emits → `len(shim) == 64` (deque maxlen).
   - `emit` is awaitable (returns a coroutine).
</behavior>

<implementation>
1. Create `src/vibemix/coach/citation_ipc_shim.py` with the class above. Pure
   stdlib (`collections.deque`); no async I/O. Doc-string explains that it's
   the v2.0 anti-slop kwarg-satisfying shim — v2.x replaces with a real
   multiplexed WS emitter.
2. Update `src/vibemix/coach/__init__.py` to re-export `CitationIpcShim`.
3. In `__main__.py`, after the linter primitives block from task 20-05-01,
   add:
   ```python
   citation_shim: CitationIpcShim | None = CitationIpcShim() if anti_slop_enabled else None

   def _citation_telemetry() -> dict:
       """Closure read fresh per publish-gate tick (every 2s in coach_loop).
       T-20-05-03: callable MUST NOT raise — coach_loop wraps in try/except
       but the dict shape is what the SessionCitation.make() factory expects."""
       reg_tel = evidence_registry.citation_telemetry()
       mean = reg_tel["mean"]
       # Placeholder slop_ratio: 1/(1+mean) — drops to 0 as citation count
       # rises. v2.x replaces with a real slop-vs-clean turn ratio.
       slop_ratio = 1.0 / (1.0 + mean) if mean > 0 else 1.0
       rate = stripped_rate_tracker.rate() if stripped_rate_tracker is not None else 0.0
       # Non-destructive bypass-readiness signal (NOT should_bypass — that
       # consumes the one-shot latch).
       bypass_active = (
           stripped_rate_tracker is not None
           and rate > 0.4  # STRIPPED_RATE_THRESHOLD — read from constants to avoid drift
       )
       return {
           "slop_ratio": float(slop_ratio),
           "stripped_rate_15s": float(rate),
           "last_unverified_response": None,  # v2.x follow-up
           "bypass_active": bool(bypass_active),
       }
   ```
4. Pass `ipc_bus=citation_shim, citation_telemetry=_citation_telemetry if
   anti_slop_enabled else None` into the `coach_loop(...)` call.
5. Import `STRIPPED_RATE_THRESHOLD` from `vibemix.coach.constants` and use
   that instead of the magic `0.4` literal in the closure (cleaner; SUMMARY
   notes the import).
</implementation>
