- `set -euo pipefail`.
- Inputs: `EAR_TEST_DIR="${EAR_TEST_DIR:-eval/ear-test-logs}"`, `WINDOW_DAYS="${WINDOW_DAYS:-14}"`, `MIN_SESSIONS=2`, `MIN_GENRES=2`.
- Logic (jq + bash date arithmetic; check `command -v jq` first, fail gracefully with clear message if missing):
  1. Compute cutoff date: `cutoff=$(date -u -v-${WINDOW_DAYS}d +%Y-%m-%dT%H:%M:%SZ)` (macOS) — also support GNU date (`date -u -d "-${WINDOW_DAYS} days" ...`). Probe + branch.

  4. Count distinct genres in the filtered set; count sessions.
  5. Count any-slop-flag-true sessions in the filtered set.
  6. Accept iff: `sessions >= MIN_SESSIONS` AND `distinct_genres >= MIN_GENRES` AND `slop_flagged == 0`.
