## Contract
Python `derive_emotion(active_genre, rms, phase, time_in_phase) -> MascotEmotion`:
- hyped: rms >= 0.18 (any genre).
- focused: active_genre in {techno, house} and 0.08 <= rms < 0.18.
- concerned: rms < 0.08 and time_in_phase >= 30 sec.
- neutral: anything else.

Frontend `EmotionLayer`: state holder. `update(emotion, now_ms)` enqueues to PriorityStack if emotion changed. Priority 60.

## Verification
- pytest: 8 cases covering all 4 emotion paths + boundary thresholds.
- vitest: EmotionLayer transitions across emotion changes; same emotion = no re-fire.
