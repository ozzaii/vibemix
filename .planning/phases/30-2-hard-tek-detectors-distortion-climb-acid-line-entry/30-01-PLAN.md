## DSP design

### `band_spectral_flatness(samples, sr, low_hz=200, high_hz=2000) -> float`
- Hanning-windowed rfft over `_SPEC_WIN` (16384).
- Restrict to `[low_hz, high_hz]` bins.
- Wiener entropy: `geometric_mean(mag) / arithmetic_mean(mag)`. Use log-sum trick for numerical stability:
  `exp(mean(log(mag + 1e-12))) / (mean(mag) + 1e-12)`.
- Returns 0.0 on silence.

### `harmonic_distortion_proxy(samples, sr, fundamental_hz=60) -> float`
- Compute rfft magnitudes.
- Identify bin positions for harmonics 1..6 of `fundamental_hz`.
- Sum magnitude at odd harmonics (3, 5) vs even (2, 4, 6).
- Return ratio `odd / (even + 1e-12)`. Distortion (especially clipping/saturation) emphasizes odd harmonics.
- Float in [0.0, inf]; baseline pure tone ‚âà 0.0 (odd harmonics empty).

## Detector design

### State
- `last_event_at: float = 0.0`
- `flatness_history: list[float]` ‚Äî last 4 samples (sliding window).
- `chain_position: int = 0` ‚Äî incremented every fire.
- `_density_streak_start: float | None = None` ‚Äî sustained kick-density tracker.

### `detect(state, audio_buf, now)`
1. Silence gate: `state.rms < LOW_RMS` ‚Üí reset density streak, return None.
2. `audio_buf is None` ‚Üí return None.
3. Cooldown gate: `now - last_event_at < 6.0` ‚Üí return None.
4. Snapshot 4s window: `audio_buf.snapshot(int(sr * 4.0))`.
5. Compute `flatness = band_spectral_flatness(samples, sr, 200, 2000)`.
6. Append to `flatness_history` (deque maxlen 4). If `len < 4` ‚Üí return None.
7. Compute `flatness_delta = flatness_history[-1] - flatness_history[0]`. Require `>= 0.15`.
8. Compute `distortion_ratio = harmonic_distortion_proxy(samples, sr, 60)`. Require `>= 1.5`.
9. Kick density gate: `state.onset_density >= 8.0`. Track sustained start:
   - If `state.onset_density >= 8.0` and `_density_streak_start is None`: set `_density_streak_start = now`.
   - If `state.onset_density < 8.0`: clear `_density_streak_start`.
   - Require `now - _density_streak_start >= 4.0`.
10. ALL three conditions met ‚Üí fire:
    - `chain_position += 1`
    - `distortion_db = 20 * log10(distortion_ratio)` (with floor for inf-safety).
    - `Event("DISTORTION_CLIMB", state, extra={"chain_position": self.chain_position, "distortion_db": round(distortion_db, 2)})`
    - `last_event_at = now`
    - Clear flatness_history (force fresh accumulation post-fire).
    - Return Event.

## Risks / mitigations
- **Risk:** Synthetic signal may not reliably cross flatness threshold. **Mitigation:** Test uses white-noise burst combined with carrier ‚Äî flatness will rise from ‚âà 0.0 (sine) to > 0.5 (noise-dominant).
