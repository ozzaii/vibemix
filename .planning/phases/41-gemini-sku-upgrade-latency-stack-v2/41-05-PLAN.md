<probe_design>
<!-- Probe behavior + integration semantics. -->

`_probe_ga_model_id(client) -> tuple[str, str]` returns `(model_id, excerpt_strategy_version)`:

```
GA_CANDIDATES = ("gemini-embedding-002", "gemini-embedding-2")
VERSION_FOR_GA_RENAME = "v2-3excerpt-mean-emb2-ga"
VERSION_DEFAULT = "v1-3excerpt-mean"

def _probe_ga_model_id(client) -> tuple[str, str]:
    canary = "vibemix probe"
    for candidate in GA_CANDIDATES:
        try:
            result = client.models.embed_content(
                model=candidate,
                contents=[types.Part.from_text(text=canary)],
                config=types.EmbedContentConfig(output_dimensionality=768),
            )
            if result and result.embeddings:
                version = VERSION_FOR_GA_RENAME if candidate == "gemini-embedding-002" else VERSION_DEFAULT
                return candidate, version
        except Exception:
            continue
    # All candidates failed — surface error (caller logs); preserve current shape
    raise RuntimeError(
        "All GEMINI_EMBEDDING_MODEL_GA_CANDIDATES failed probe. "
        "Embeddings unavailable until network / API restored."
    )
```

Integration into LibraryEmbedder:
- `__init__` calls `_probe_ga_model_id(self._client)` ONCE at construction; caches the result on the instance.
- If `EXCERPT_STRATEGY_VERSION` returned is the GA-rename version, the cache key changes for every track → next call naturally re-embeds (existing lazy path).
- Probe runs ONCE per process — not per call.
- Emits `embedding_model_probe` event with `{candidate_tried: [...], chosen: "...", version: "...", duration_ms: ...}`.
</probe_design>
