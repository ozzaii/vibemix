**Class `LibraryEmbedder`:**
- `__init__(self, client: genai.Client, cache_db: sqlite3.Connection | None = None)` — accepts a pre-built proxy client (built via `build_proxy_genai_client` upstream) + optional cache DB connection. Default cache DB = open `EMBED_CACHE_DB_PATH` (mkdir parents). Schema: `CREATE TABLE IF NOT EXISTS embed_cache (key TEXT PRIMARY KEY, vector BLOB NOT NULL, ts REAL NOT NULL)`.
- `embed_track(self, track: TrackEntry) -> np.ndarray` — single entry point. Decision tree:
  1. Compute `key = _track_hash(track)`. Check cache → return cached vector if hit.
  2. If `Path(track.filepath).exists()` AND ffmpeg available → audio path (3-excerpt for `duration_s > 180`, single call for `duration_s <= 180`).
  3. Else → text-only path: embed `f"{track.title} by {track.artist} | {int(track.bpm or 0)} BPM | key {track.key or 'unknown'}"`.
  4. Wrap returned vector with `l2_normalize` from `_cosine`. Persist via `INSERT OR REPLACE INTO embed_cache`. Return.
- `embed_query(self, query: str) -> np.ndarray` — text-only path for vibe-search. Used by Plan 03. No content-hash cache here (the 24h query cache lives at Plan 03 level, keyed on `query + library_snapshot_hash`). Returns L2-normalized float32 vector.
- `_track_hash(track: TrackEntry) -> str` per RESEARCH Pattern 3: SHA256 of `(file_bytes_or_marker || GEMINI_EMBEDDING_MODEL || EXCERPT_STRATEGY_VERSION)`. For streaming-only (no local file): use `f"<streaming>{track.track_id}".encode()` as the file portion. For local files: stream in 64KB chunks.
- `_extract_excerpts(audio_path: Path, duration_s: float) -> list[bytes]` — uses ffmpeg subprocess (verify availability with `shutil.which("ffmpeg")` at boot; raise `RuntimeError("ffmpeg required")` if absent per RESEARCH Pitfall: "ffmpeg not available", fail-loud at import-start). Slice 3 mp3 excerpts: intro `0-60`, mid `duration_s/2 - 30 to duration_s/2 + 30`, outro `max(0, duration_s-60)` to end. Use ffmpeg `-ss <start> -t 60 -acodec libmp3lame -b:a 128k` to a tempfile, read bytes, delete tempfile. For tracks ≤ 180s return `[audio_path.read_bytes()]` (single call path; bypass ffmpeg). FLAC/AAC source MIME notes per Open Q6: ffmpeg transcodes to MP3 inline → upload as `audio/mpeg`. If audio file is already a WAV, pass `audio/wav` instead.
- `_call_gemini_audio(self, clips: list[bytes]) -> list[np.ndarray]` — wraps `client.models.embed_content` for each clip with `types.Part.from_bytes(data=clip, mime_type='audio/mpeg')`, `config=types.EmbedContentConfig(output_dimensionality=EMBEDDING_DIM)`. Try/except: catch `google.genai.errors.APIError` (or generic `Exception` if exact class unclear) when message contains `"too long"` or HTTP 400 with audio cap → log + caller falls back to per-excerpt path. Return list of `np.asarray(result.embeddings[0].values, dtype=np.float32)`.
- `_call_gemini_text(self, text: str) -> np.ndarray` — `client.models.embed_content(model=GEMINI_EMBEDDING_MODEL, contents=text, config=types.EmbedContentConfig(output_dimensionality=EMBEDDING_DIM))`. NOT `contents=[text]` — text path takes a string directly per Gemini SDK 2.0.1.

**Average for 3-excerpt path:** `np.mean(np.stack(vectors), axis=0).astype(np.float32)` then `l2_normalize`. The cache stores the L2-normalized averaged vector (callers can dot-product directly).

**Per D-cost-balanced (CONTEXT):** `output_dimensionality=768`. Comment notes 3072 supported via MRL but rejected for storage cost.

| Boundary | Description |

| User file system → embed.py | Audio file bytes (potentially large, attacker-controlled if a malicious XML/folder is dropped) are SHA256'd + uploaded. |
| Cache DB writes | `~/.cache/vibemix/embeddings.db` — write-by-sidecar-only, user-owned. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
