<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| ws://127.0.0.1:8766 → renderer | Plan 02 sidecar (trusted local) sends JSON; ajv-compiled validator gates dispatch. |
| URL ?session= param → renderer | The query-string came from Rust open_debrief_window — already validated by Plan 04 validate_under_root. Defense-in-depth: renderer skips load if param missing/empty. |
| asset://<session_dir>/voice.wav → WaveSurfer | Phase 15 capability scope (`recordings/**`) gates the URL; renderer can't load arbitrary FS paths. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-29-05-01 | Tampering | Malformed WS frame triggers component crash | mitigate | Validate every frame via ajv before dispatch. Invalid frames logged + ignored. |
| T-29-05-02 | Information Disclosure | Citation tooltip text contains session data — could leak via screenshot | accept | Same trust level as the rest of the app — user is local single-user. |
| T-29-05-03 | Denial of Service | Renderer drowns in chapter regions on a session with 1000+ events | mitigate | derive_chapters (Plan 01) already collapses HEARTBEAT + < 30s deltas; chapter count is bounded ~ session_duration_min. |
| T-29-05-04 | Spoofing | Renderer trusts any process bound to 127.0.0.1:8766 | accept | Local-only single-user app; same model as main ws_bus on 8765. |
| T-29-05-05 | Tampering | session_dir injected via URL crafting | mitigate | Plan 04 validates before window open; renderer treats param as opaque string never path-joined for FS access — only passed to convertFileSrc which Tauri-side validates against asset scope. |
| T-29-05-06 | Tampering | XSS via drill text rendering | mitigate | Use `.textContent` not `.innerHTML` for all dynamic strings. Citation chips render the tag literal via textContent. |
| T-29-05-07 | Information Disclosure | TLDR audio is fetched but never played — bandwidth waste on remote sidecars | accept | Local sidecar, asset:// is local-disk — no bandwidth concern. |
</threat_model>
