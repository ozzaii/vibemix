OPUS decode pattern:
```python
import av, numpy as np
with av.open(opus_path) as container:
    stream = next(s for s in container.streams if s.type == "audio")
    pcm_chunks = []
    for frame in container.decode(stream):
        # frame is av.AudioFrame; convert to ndarray
        arr = frame.to_ndarray()  # shape (channels, samples), dtype varies
        pcm_chunks.append(arr)
    pcm = np.concatenate(pcm_chunks, axis=-1)  # (channels, total_samples)
# Resample to 24kHz int16 mono if needed (av.AudioResampler):
resampler = av.AudioResampler(format="s16", layout="mono", rate=OUTPUT_SR)  # 24000
```
