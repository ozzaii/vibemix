winsdk API surface (verified via Microsoft docs, https://github.com/pywinrt/python-winsdk):
```python
import winsdk.windows.media.control as wmc

async def get_current_title() -> tuple[str, str] | None:
    mgr = await wmc.GlobalSystemMediaTransportControlsSessionManager.request_async()
    session = mgr.get_current_session()
    if session is None:
        return None
    props = await session.try_get_media_properties_async()
    return (props.artist or "", props.title or "")
```
Note: winsdk returns awaitables that integrate with asyncio. CONTEXT recommends bridging via `asyncio.run_in_executor(None, lambda: asyncio.run(coroutine))` — wraps the entire async call in a sync executor thread, matches macOS subprocess pattern.

      try:
          import winsdk.windows.media.control as wmc
      except ImportError:
          return None

      async def _inner():
          mgr = await wmc.GlobalSystemMediaTransportControlsSessionManager.request_async()
          session = mgr.get_current_session()
          if session is None:
              return None
          props = await session.try_get_media_properties_async()
          artist = (props.artist or "").strip()
          title = (props.title or "").strip()
          if artist and title:
              return f"{artist} - {title}"
          return artist or title or None

      try:
          return asyncio.run(_inner())
      except Exception as e:
          if not self._state._has_logged_unavailable:
              print(f"-> SMTC unavailable: {e}", file=sys.stderr)
              self._state._has_logged_unavailable = True
