## 1000-cycle race test

```python
import threading
import pytest

from vibemix.state.genre_router import GenreRouter

def test_genre_router_1000_cycle_concurrent_swap_no_race():
    """8 reader threads iterate active_chain() while main thread swaps
    between hard_tek and techno 1000 times. No raise, no half-state."""
    router = GenreRouter()
    router.swap("hard_tek")

    stop = threading.Event()
    errors: list[BaseException] = []

    def reader():
        try:
            while not stop.is_set():
                chain = router.active_chain()
                # iterate it the way EventDetector.detect does
                for _ in chain:
                    pass
        except BaseException as e:
            errors.append(e)

    threads = [threading.Thread(target=reader, daemon=True) for _ in range(8)]
    for t in threads:
        t.start()

    try:
        for i in range(1000):
            router.swap("techno" if i % 2 else "hard_tek")
    finally:
        stop.set()
        for t in threads:
            t.join(timeout=5.0)

    assert not errors, f"Race test surfaced errors: {errors!r}"
    # Final state must be one of the two genres we swapped to.
    assert router.current_genre in ("techno", "hard_tek")
```

## Immutability test

```python
import pytest
from vibemix.events.genres import GENRE_REGISTRY

def test_genre_registry_is_immutable_mapping():
    """P49 mitigation — GENRE_REGISTRY is MappingProxyType. Attempts to
    register a chain at runtime raise TypeError."""
    with pytest.raises(TypeError):
        GENRE_REGISTRY["dubstep"] = lambda: []  # type: ignore[index]
```

## Risks / mitigations
- **Risk:** Test flake on slow CI. **Mitigation:** 8 threads × 1000 swaps completes in ~50ms locally; generous join timeout 5s.
- **Risk:** `MappingProxyType[str, Callable]` generic syntax not supported on older Python. **Mitigation:** Project uses Python 3.14 (CLAUDE.md) / 3.12 (phase_state); both support PEP 696 generic syntax for `MappingProxyType` via `from __future__ import annotations` lazy resolution. If generic subscript fails, fall back to bare `MappingProxyType`.
