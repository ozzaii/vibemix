    situation: str
    behavior: str
    impact: str
    action_recommended: str
    citation: str  # MUST be a [ev:*] / [track:*] / [mix:*] tag present in evidence_snapshot

class Drills(BaseModel):

@dataclass(frozen=True, slots=True)
class ChapterRegion:
    id: str           # stable: e.g., "track-01", "phase-build-04:32"

    end: float
    label: str        # human-visible
    kind: str         # one of: track | phase | layer | mix | crowd

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Gemini API → debrief module | Untrusted text/JSON from external LLM crosses here; Pydantic responseSchema + regex citation validation guard it. |
| events.jsonl → chapters.py | Local file already authored by VoiceRecorder (trusted), but malformed JSON lines must not crash parser. |
| evidence_registry.json → drills.py / stripper.py | Same VoiceRecorder author; treated as trusted local data. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-29-01-01 | Tampering | Gemini emits drill citing event that doesn't exist | mitigate | Drill citation validated against evidence_snapshot at ±2.0s; unresolvable drills dropped + re-requested (RESEARCH Pitfall #7). |
| T-29-01-02 | Information Disclosure | Gemini API call logs full session text to console/log | mitigate | Reuse Phase 27 LATENCY-15 pattern — log only `OK <session>/debrief_tldr.mp3 (<n> bytes)`; never log response repr. |
| T-29-01-03 | Tampering | events.jsonl line is malformed JSON | mitigate | Parser wraps each `json.loads` in try/except; bad lines skipped + warning logged; chapter derivation continues. |
| T-29-01-04 | Denial of Service | Gemini API hangs indefinitely | mitigate | Timeout set on Gemini client (30s for TLDR text, 60s for drills, 90s for TTS audio); on timeout emit `ipc.debrief.error` (Plan 02 surfaces). |
| T-29-01-05 | Spoofing | Drill citation looks valid but references unrelated evidence | accept | Citation tag is a string match — if attacker controls evidence_snapshot they already control the session. No new surface. |
| T-29-01-06 | Repudiation | TLDR audio cached forever — user can't tell when it was generated | mitigate | session_debrief.json records `generated_at` ISO timestamp + `tldr_sha256` hash; UI shows "Generated 2026-05-15" via Plan 05. |
| T-29-01-07 | Information Disclosure | TTS audio uploads session text to Gemini API | accept | Gemini-only architecture is a design lock (memory `feedback_no_clap_use_gemini_embedding`); proxy mode keeps API key out of client. |
</threat_model>
