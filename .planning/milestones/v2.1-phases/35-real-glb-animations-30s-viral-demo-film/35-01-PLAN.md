# Plan 35-01 — GLB Optimization Pipeline

**REQ-IDs:** ASSETS-04
**Pitfall:** P52

## Goal

Ship `scripts/glb_optimize.py` — a gltfpack-based DRACO L7 + KTX2 batch optimizer with two modes:
- `--optimize INPUT_DIR OUTPUT_DIR` — run gltfpack on every `.glb` in INPUT_DIR. Requires gltfpack on PATH.
- `--check ROOT_DIR` — scan existing `.glb` files; assert per-clip ≤ 600 KB and total ≤ 25 MB. No external deps.

`--check` is the CI mode; `--optimize` is Kaan-action local mode.

## Tasks

1. Write `scripts/glb_optimize.py`:
   - argparse with two mutually-exclusive modes.
   - `--check` walks ROOT_DIR recursively for `*.glb`, asserts per-clip ≤ 600 KB, total ≤ 25 MB. Prints per-clip table on failure. Exit 1 on violation.
   - `--optimize` shells out to `gltfpack -i {in} -o {out} -cc -tc -tq 8`. If gltfpack missing → exit 2 with install hint.
   - Constants `MAX_PER_CLIP_BYTES = 600 * 1024` and `MAX_TOTAL_BYTES = 25 * 1024 * 1024` at module top.

2. Write `tests/scripts/test_glb_optimize.py`:
   - `test_check_passes_under_budget` — synthetic 500 KB file in tmpdir → exit 0.
   - `test_check_fails_per_clip_over_600kb` — synthetic 700 KB file → exit 1, stderr names file.
   - `test_check_fails_total_over_25mb` — many synthetic files summing > 25 MB → exit 1, stderr names total.
   - `test_check_empty_dir_passes` — no GLBs → exit 0.
   - `test_optimize_mode_missing_gltfpack_exits_2` — monkeypatch PATH to omit gltfpack → exit 2.
