Class `LibraryImporter`:
- `__init__(self, embedder: LibraryEmbedder, store: LibraryStore, ws_bus, batch_size: int = 10)`. The `ws_bus` is the Phase 11 ws_bus (port 8765); used to emit `ipc.library.import_progress` ticks. If injecting `ws_bus` is awkward (architecture varies — check `runtime/ws_bus.py` shape), accept a callable `on_progress: Callable[[dict], None]` instead.
- `self.cancel_flag = asyncio.Event()` — set by the IPC handler on `ipc.library.import_cancel`.
- `async def import_library(self, xml_path: Path) -> dict`:
  1. `lib = RekordboxLibrary(); lib.load_xml(xml_path)` (Phase 25 API).
  2. Persist updated `library.pkl` to `~/.cache/vibemix/library.pkl` (so the Plan 04 register_library wiring picks up tracks at next boot).
  3. `total = len(lib.tracks)`.
  4. `done = 0; cache_hits = 0`.
  5. Iterate tracks in batches of `batch_size`:
     - For each track: if `self.cancel_flag.is_set()`: emit final progress tick (`done`, `cache_hits`, `cancelled=True`) and break.
     - `vec = await loop.run_in_executor(None, self.embedder.embed_track, track)`. Catch exceptions per-track: log + skip + continue (one bad audio file should not block the batch).
     - Track whether this was a cache hit: instrument the embedder to expose a `last_was_cache_hit` flag, OR query the cache directly (cheaper to instrument).
     - Append `(track.track_id, vec)` to batch buffer.
     - Increment `done`. If cache hit, increment `cache_hits`.
     - Emit `ipc.library.import_progress` payload: `{total, done, current_track_name: f"{track.title} — {track.artist}", cache_hits}` via `on_progress` callback.
   - At batch boundary: `self.store.add_batch(batch_buffer)` (atomic write).
  6. After all batches: emit final progress + return `{total, done, cache_hits, cancelled: self.cancel_flag.is_set()}`.

Function `import_library_async(xml_path, ws_bus, evidence_registry) -> dict`: orchestrator. Builds embedder + store, runs `LibraryImporter.import_library`. Returns the final dict (rendered to UI as the import done state). After import: call `evidence_registry.register_library(lib)` to refresh the in-memory citation registry without requiring a session restart (improves UX — citations work mid-session).

The IPC dispatcher (in `runtime/ws_bus.py` or `ui_bus/messages.py`) routes:
- `ipc.library.import` → `import_library_async(payload.path, ...)` (fire-and-forget asyncio task)
- `ipc.library.import_cancel` → `importer.cancel_flag.set()` on the active importer instance.
