```python
STALE_AGE_SECONDS = 30 * 86400
def is_stale(library_pkl: Path = None) -> tuple[bool, int]:
    """Returns (stale_flag, age_days). False/0 if no library loaded."""
    library_pkl = library_pkl or Path.home() / ".cache" / "vibemix" / "library.pkl"
    if not library_pkl.exists(): return False, 0
    mtime = os.path.getmtime(library_pkl)
    age = time.time() - mtime
    return age > STALE_AGE_SECONDS, int(age // 86400)
```

- `load_snooze_state(state_path: Path | None = None) -> float | None` — read JSON, return `data.get(STATE_KEY)`. Returns `None` on FileNotFoundError or JSON parse error (graceful — better to over-nudge than under-nudge).
- `save_snooze_state(snoozed_until_ts: float, state_path: Path | None = None) -> None` — read existing JSON (or `{}`), set `data[STATE_KEY] = snoozed_until_ts`, atomic write (`tempfile` + `os.replace`). Mkdir parents.
- `is_snoozed(state_path: Path | None = None) -> bool` — returns `load_snooze_state() is not None and load_snooze_state() > time.time()`.
- `emit_nudge_if_stale(emit_ipc: Callable[[str, dict], None], library_pkl: Path | None = None, state_path: Path | None = None) -> bool`:
  1. `stale, age_days = is_stale(library_pkl)`.
  2. If not stale: return False.
  3. If `is_snoozed()`: return False (snooze active).
  4. Else: call `emit_ipc("ipc.library.staleness_nudge", {"age_days": age_days, "snoozed_until_ts": None, "schema_version": "1"})`. Return True.

- `apply_snooze_action(action: Literal["dismiss", "snooze_7d"], state_path: Path | None = None) -> None`:
  - `"dismiss"`: no-op on disk (banner hides this-session only via UI state).
  - `"snooze_7d"`: `save_snooze_state(time.time() + SNOOZE_DURATION_SECONDS, state_path)`.

Update `library/__init__.py`: re-export `is_stale`, `emit_nudge_if_stale`, `apply_snooze_action`, `STALE_AGE_SECONDS`, `SNOOZE_DURATION_SECONDS`.

**Atomic write** for state.json: write to `state.json.tmp` then `os.replace(tmp, state_path)`. Prevents corruption on power loss mid-write.

**Defensive parse**: if `state.json` exists but is malformed JSON, log a warning and treat as `{}` (don't crash boot path).
