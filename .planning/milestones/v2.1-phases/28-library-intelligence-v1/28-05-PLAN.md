  <files>src/vibemix/library/similar.py, src/vibemix/library/__init__.py</files>
  <action>
Create `src/vibemix/library/similar.py`.

Module constants:
- `SIMILARITY_DEFAULT_K = 10`

Dataclass `SimilarTrack` (frozen, slots):
- `track_id: str`
- `similarity: float` (cosine in [0, 1], rounded to 4 decimals)
- `title: str`
- `artist: str`
- `bpm: float | None`

Function `find_similar(store: LibraryStore, library: RekordboxLibrary, track_id: str, k: int = SIMILARITY_DEFAULT_K) -> list[SimilarTrack]`:
1. `ids, vectors = store.load_all()`.
2. If `track_id not in ids`: raise `KeyError(f"track_id '{track_id}' not in library ŌĆö re-import or check ID")`.
3. Get the row index, extract the query vector: `qvec = vectors[ids.index(track_id)]`.
4. Run `top = cosine_topk(qvec, vectors, ids, k=k + 1)` (k+1 because the named track itself will rank #1 at cosine 1.0 ŌĆö strip it).
5. Filter out the named track from results (since `qvec` is its own vector, it will appear first at cosine ~1.0).
6. Resolve each (tid, sim) ŌåÆ SimilarTrack via library tracks dict.
7. Return top-k SimilarTrack list. If library has fewer than k other tracks, return what exists.

**Anti-feature guard contract (encoded in code):**
- The module MUST NOT import `asyncio`, MUST NOT register any subscriber, MUST NOT export any class with background tasks. It is a pure function module.
- Add a module docstring banner stating the anti-feature contract: "This module is invoked ONLY on explicit user request (CLI / IPC). No auto-emission, no event subscription, no scheduler."

Update `src/vibemix/library/__init__.py`: re-export `find_similar`, `SimilarTrack`, `SIMILARITY_DEFAULT_K`.
