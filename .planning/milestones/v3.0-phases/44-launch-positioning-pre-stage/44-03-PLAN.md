<objective>
Surface the existing `EvidenceRegistry` citations as user-visible chips in the live session UI: every AI reaction in the cohost stream gets a `[<verb> @ <mm:ss>]` chip; clicking a chip opens the debrief window with that timestamp highlighted on the waveform timeline. Closes the §6.2 white-space gap where the anti-slop primitive existed in the backend but was invisible to users.

Purpose: LAUNCH-02. This is the anti-slop receipt that vibemix shows on screen. The hero promise ("the only AI co-host that actually listens to your set") is grounded for the user by visual proof: every reaction is tagged with the exact event that triggered it. Click-through to debrief turns the receipt into a learning loop — "what did the AI actually hear at 2:33?" → waveform region opens, the user can replay, the credibility lock activates.

Output: New `citation-strip.ts` component wired into the session cohost stream; backend `citation_strip` field emitted on reaction broadcast; Tauri IPC + debrief deep-link wiring; tokens-driven CSS; full test coverage on both ends. Wave 1 because it doesn't touch README.md (no serialization with 44-01 / 44-02).
</objective>
