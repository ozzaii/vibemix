## Why deferred

Free VB-CABLE covers the v3.1 Windows install path end-to-end. Voicemeeter Banana is VB-Audio's donationware tier above VB-CABLE — it adds multi-bus routing useful for streamers running multiple audio sources at once. Banana is an advanced-user surface that does not need installer bundling.

## Re-eval trigger

- User feedback shows demand for multi-source routing on Windows that VB-CABLE alone cannot serve.
- A v3.x phase scopes a streamer-multi-bus tutorial as a goal.

## Context

If demand materializes, Banana lands as a documented optional path in the calibration wizard or a docs/integrations/ page, not as a runtime dep. The v3.1 baseline ships VB-CABLE only.
