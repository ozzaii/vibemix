## Why deferred

Apple has not shipped the next-major macOS as of v3.1 close. The current Phase 33 v2.1 ladder covers macOS 12.3 / 14 / 15; Phase 50 SHIP-04 fresh-VM matrix targets those versions. Bumping the matrix before the OS ships is premature.

## Re-eval trigger

- Apple ships the next-major macOS publicly.
- vibemix needs SHIP-04 matrix expansion (CI VM image + smoke runs).

## Context

ScreenCaptureKit, MediaRemote (nowplaying-cli), and CoreAudio behaviors typically shift on macOS major bumps. The v3.x phase that lands this surface should run the existing Phase 50 SHIP-04 smoke harness against the new OS image and patch any drift surgically.
