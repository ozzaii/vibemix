# Third-Party Fonts — Vendored Under Their Respective Licenses

vibemix ships 4 WOFF2 fonts bundled under `tauri/ui/public/fonts/` (1 Saira
variable + 3 JetBrains Mono static). Vendoring them locally (rather than
fetching from Google Fonts at runtime) is a hard requirement so the wizard
renders correctly on first launch when the user has no network — and so
production codesigning in Phase 18 can attest the exact binary contents.

Each entry below records source, license, and SHA-256 of the WOFF2 we ship.
Vendor swaps MUST update the SHA here in the same PR (Phase 18 codesign chain
+ the UI-checker (Phase 14) re-verification depend on stable hashes).

Phase 14 Wave 5 (commit landing this file change) replaced the four legacy
families used in the FL-Studio retro-tactile prototype — Workbench, DM Mono,
DSEG7 Classic, Caveat — with two CDJ-Whisper-v5 families: Saira (variable
width + weight) for chrome, JetBrains Mono (three static weights) for
numerics. The legacy WOFF2 files were deleted in the same commit.

## vibemix mascot — Neon Rebel (Meshy AI generated, Kaan-owned)

- **Files:**
  - `tauri/ui/assets/mascot/character.glb` (base rigged character + texture)
  - `tauri/ui/assets/mascot/animations/*.glb` (20 single-clip animation GLBs)
  - `tauri/ui/assets/mascot/manifest.json` (clip → state-vocab map)
  - `tauri/ui/assets/mascot/MANIFEST.md` (compression report)
- **Source:** [Meshy AI](https://www.meshy.ai/) — character + animation library
  generated by Kaan from text prompts on the Meshy AI platform. Raw exports
  live OUTSIDE this repo at `/Users/ozai/Downloads/Meshy_AI_Neon_Rebel_biped/`
  and are NOT redistributed; only the compressed bundle ships in the app.
- **License:** Kaan-owned generated content per [Meshy AI Terms of Service](https://www.meshy.ai/legal/terms).
  Meshy's ToS grants the generator (Kaan / OZAI) the right to redistribute
  generated assets as part of a software product, including for commercial
  use, with no attribution requirement. We attribute Meshy AI here for
  transparency, not as a license obligation.
- **Compression pipeline:** `tauri/ui/scripts/build-mascot-bundle.mjs` →
  `gltf-pipeline` Draco (character) + `@gltf-transform/core` strip
  (animations). See `tauri/ui/assets/mascot/MANIFEST.md` for per-file
  compression ratios and the locked Draco flags.
- **SHA-256:** not pinned — assets are Kaan-generated and refreshed in-place
  when Meshy outputs improve. The committed tree IS the source of truth for
  reproducible builds; CI does not re-run `build:mascot`.
- **Use in vibemix:** Phase 13 3D mascot screen-overlay window. Three.js
  loads `character.glb`, then extracts the `AnimationClip` from each
  `animations/*.glb` and binds it to the character's skeleton via
  `SkeletonUtils.retargetClip` (Plan 13-04).

## SIL Open Font License 1.1 — Summary

All four vendored fonts ship under the SIL Open Font License 1.1. The OFL
permits redistribution as part of a software product so long as the font
files themselves are not sold as a standalone product and the font name is
not changed. Both conditions hold here:

1. Fonts ship as part of the vibemix application — not sold separately.
2. Filenames preserve upstream family names (`Saira-VariableFont_wdth,wght`,
   `JetBrainsMono-Regular`, `JetBrainsMono-Medium`, `JetBrainsMono-SemiBold`).

Full OFL 1.1 text: https://scripts.sil.org/OFL_web
